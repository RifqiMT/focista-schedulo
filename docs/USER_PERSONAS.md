# User Personas — Focista Schedulo

**Last updated**: 2026-03-18  
**Owner**: Product  

## Persona 1 — The Project Operator

- **Profile**: mid-level professional managing multiple deliverables, meetings, and cross-team dependencies.
- **Primary goals**:
  - Keep tasks organized by project and priority.
  - Plan the day realistically based on duration and meetings.
  - Quickly move tasks when scope changes.
- **Key behaviors**:
  - Creates tasks in batches for a project.
  - Uses calendar to decide what fits today vs tomorrow.
  - Tracks completion streaks for motivation.
- **Pain points**:
  - “To-do lists” become noisy without a calendar view.
  - Recurring admin tasks duplicate or drift over time.
  - Editing recurring series is often unreliable.
- **What success looks like**:
  - One view to see the month and drill into a day agenda.
  - Recurring tasks stay stable and predictable.
  - Bulk operations make maintenance easy.

## Persona 2 — The Routine Builder

- **Profile**: student or learner building habits (study, exercise, language practice).
- **Primary goals**:
  - Set up recurring tasks and only deal with “the next one”.
  - Capture tasks quickly with voice.
  - Feel rewarded for consistency.
- **Key behaviors**:
  - Uses “weekdays/weekends” and custom intervals (every 2 weeks).
  - Completes tasks on mobile/laptop quickly.
  - Checks progress daily.
- **Pain points**:
  - Recurrence systems generate multiple upcoming tasks (overwhelming).
  - Voice capture often fails to fill structured fields.
- **What success looks like**:
  - Simple recurring setup; only one upcoming occurrence shown.
  - Voice input correctly fills date/time/duration/priority.

## Persona 3 — The Personal Planner

- **Profile**: individual managing life admin, errands, and personal goals.
- **Primary goals**:
  - Remember deadlines and reminders.
  - Tag tasks with labels and locations.
  - Use calendar to avoid overloading weekends.
- **Key behaviors**:
  - Uses reminders and deadlines for time-sensitive tasks.
  - Switches between list and calendar views.
  - Exports data occasionally for backup.
- **Pain points**:
  - Hard to see multi-day tasks and time blocks clearly.
  - Loses confidence if IDs/recurrence behave inconsistently.
- **What success looks like**:
  - Calendar reflects multi-day duration correctly.
  - Export is one click with format choice.
